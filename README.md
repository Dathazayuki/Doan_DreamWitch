# Dream Witch - Unity Project

Bản lưu trữ (repository) này chỉ chứa mã nguồn, prefabs và các tài nguyên cốt lõi nằm trong thư mục `Assets/Project` để giảm thiểu dung lượng lưu trữ của Git. 

Để chạy và phát triển dự án này trong Unity, vui lòng thực hiện theo các bước hướng dẫn thiết lập dưới đây:

## Hướng dẫn thiết lập dự án (Setup Instructions)

1. **Tải về các thư mục tài nguyên gốc (Assets, ProjectSettings, Packages):**
   - Vui lòng tải xuống tệp tin nén của dự án từ liên kết Google Drive sau:
     [Tải về file tài nguyên dự án (Google Drive)](https://drive.google.com/file/d/15X3Vupiz5q4MRRO6-ji1ZNghxa6Nofey/view?usp=sharing)

2. **Giải nén tệp tin đã tải:**
   - Giải nén tệp zip vừa tải về vào máy tính của bạn để tạo ra cấu trúc thư mục dự án Unity cơ sở (bao gồm các thư mục như `Assets`, `ProjectSettings`, `Packages`).

3. **Sao chép thư mục Git này vào dự án:**
   - Sao chép toàn bộ thư mục `Project` (chứa mã nguồn và tài nguyên từ repository này) và ghi đè vào thư mục `Assets/` của dự án Unity cơ sở vừa giải nén (vị trí đích sẽ là `Assets/Project`).

4. **Mở dự án trên Unity:**
   - Sử dụng **Unity Hub** để mở thư mục dự án vừa thiết lập (phiên bản Unity tương thích với dự án).
